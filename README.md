# number-guessing-game
A simple console-based Number Guessing Game developed in Java. The Program generates a random number between 1 and 100, and the user tries to guess it. The game provides feedback if the guess is too low, too high, or correct. Includes input validation to handle non-numeric entrie. 

Tools :
* Java
* Visual Studio code
  
Features :
* Generates a random number between 1 and 100
* User enters guesses in the console
* Provides hints: "Too Low" or "Too high"
* Displays"Correct guess"when the user guesses the number
* Handles invalid input(non-numeric entries)
